(1) Data visualization is a term that covers all the tools and techniques used to visualize data to help people to understand them better. It is useful to find patterns, trends and correlations.
(2) Data visualization is a communication tool that combines art and science to illustrate ideas to the audience quicker in a more straight-forward way.
(3) Data visualization is one of the steps in data science, it involves utilizing computer software packages to help create graphs to help explaining the data.
